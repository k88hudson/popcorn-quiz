Help

To create a quiz, drag a quiz element from the 'My Events' panel to the timelime. You will have to set the question, the answers, the correct answer, and an explanation for the correct answer.

Highlighting quiz answers

Drag a answer event to the track at the time you want it to be highlighted.

Adding Media or Images

For videos, Drag a mediaspawner event to the track at the time you want it to show up.

For videos, Drag an image event to the track at the time you want it to show up.

You should set the target to be media1 if you want it to appear on the left, and media2 if you want it to appear on the right.
